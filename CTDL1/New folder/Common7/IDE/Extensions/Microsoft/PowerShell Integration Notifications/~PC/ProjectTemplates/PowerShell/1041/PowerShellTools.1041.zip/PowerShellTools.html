PowerShell Tools for Visual Studio

Visual Studio での PowerShell スクリプトおよびモジュールの編集とデバッグをサポートするには、 "PowerShell Tools for Visual Studio" 拡張機能をギャラリーからインストールしてください。

PowerShell Tools for Visual Studio は Visual Studio を拡張し、 PowerShell スクリプトおよびモジュールの作成、ビルド、デバッグ、実行、およびパッケージ化を可能にします。

注意: このページを含むプロジェクトでは、必要なツールセットをインストールするための手順のみが提供されます。
