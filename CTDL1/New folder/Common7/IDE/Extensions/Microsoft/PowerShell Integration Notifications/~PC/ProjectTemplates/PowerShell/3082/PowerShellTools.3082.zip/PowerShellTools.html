Herramientas de PowerShell para Visual Studio

Para habilitar la compatibilidad para editar y depurar módulos y scripts PowerShell en Visual Studio, instale la extensión de "Herramientas de PowerShell para Visual Studio" de la Galería.

Las herramientas de PowerShell para Visual Studio amplían Visual Studio para habilitar la creación, la compilación, la depuración, la ejecución y el empaquetado de módulos y scripts PowerShell.

Nota: El proyecto que incluye esta página solo proporciona instrucciones para instalar el conjunto de herramientas requerido.
