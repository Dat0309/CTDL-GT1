PowerShell Tools for Visual Studio

Per abilitare il supporto per la modifica e il debug di script e moduli di PowerShell in Visual Studio, installare l'estensione "PowerShell Tools for Visual Studio" da Visual Studio Gallery.

Grazie a PowerShell Tools for Visual Studio è possibile usare Visual Studio per la creazione, la compilazione, il debug, l'esecuzione e la creazione di pacchetti di script e moduli di PowerShell.

Nota: il progetto che contiene questa pagina fornisce istruzioni solo per l'installazione del set di strumenti richiesto.
