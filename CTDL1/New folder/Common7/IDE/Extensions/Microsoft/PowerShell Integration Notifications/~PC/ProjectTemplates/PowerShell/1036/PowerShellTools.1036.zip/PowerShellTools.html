Outils PowerShell pour Visual Studio

Pour activer la prise en charge de la modification et du débogage des scripts et modules PowerShell dans Visual Studio, installez l'extension « Outils PowerShell pour Visual Studio » à partir de la galerie.

Les outils PowerShell pour Visual Studio étendent les capacités de Visual Studio pour permettre la création, la génération, le débogage, l'exécution et l'empaquetage des scripts et modules PowerShell.

Remarque : le projet qui contient cette page fournit uniquement des instructions d'installation de l'ensemble d'outils nécessaire.
