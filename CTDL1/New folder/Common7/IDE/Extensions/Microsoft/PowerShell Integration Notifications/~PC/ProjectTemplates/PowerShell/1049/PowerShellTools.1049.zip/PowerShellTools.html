Инструменты PowerShell для Visual Studio

Чтобы включить поддержку редактирования и отладки скриптов и модулей PowerShell в Visual Studio, установите расширение "инструменты PowerShell для Visual Studio" из коллекции.

Инструменты PowerShell для Visual Studio добавляют в Visual Studio возможность создавать, выполнять сборку и отладку, запускать и упаковывать скрипты и модули PowerShell.

Примечание. Проект, содержащий эту страницу, предоставляет только инструкции по установке требуемого набора инструментов.
