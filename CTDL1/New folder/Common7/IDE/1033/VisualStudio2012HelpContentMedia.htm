Installing Product Documentation

You can quickly and easily download the most recent version of the product Help from MSDN to your local computer. This version includes all changes and new topics that have been added since the product’s release. For more information about how to install Help locally, see the Install and Manage Content topic in the Help for Microsoft Help Viewer 2.0.

If your Internet access is limited and you prefer not to download content, consider the following options:
  • Download a subset of the Help that covers just the products or technologies that you need.
  • Create an installation disk by downloading the ISO image from the download location. If you aren't familiar with this type of file, see Using ISO Image Files.
  • Order the Help on DVD from your country or region's customer support team. You will be charged a fee to cover the cost of goods and shipping. To order the DVD, call the number for your region from the following list.

Using Microsoft Help Viewer 2.0
  • The fulfillment DVD for MSDN Content contains content for Microsoft Help Viewer 2.0, which is installed with Visual Studio 2012. You can’t install this content with Microsoft Help Viewer 1.1.
  • You can’t redistribute Help Viewer 2.0, and it’s installed only with certain products in Visual Studio 2012.
  • In some installations, an administrator may have disabled the Manage Content tab. In those cases, you can’t install Help content by using the instructions in this article. You must consult your administrator.
  • To install Help content, you must have administrative permissions on the local computer.

Using ISO Image Files

Many products are posted as an ISO-9660 image file, which is an exact representation of a CD or DVD, including the content and the logical format. Users typically write image files to blank CD-Rs or DVD-Rs, resulting in identical copies of the original disk, including the file name and the volume label. You can also open an ISO image file and copy its contents to a local folder, much like a compressed .zip file. As an alternative, you can virtually mount and access an ISO file as a device. Depending on your software, you may have to rename the file extension from .img or .udf to .iso.

Your software for writing CD-Rs and DVD-Rs probably includes a "copy image to CD" or "burn image" feature that you can use to create a disc from an image file. For more information, see the Help for your software. After you create the disc, complete the appropriate set of steps in the next section.

Installing Documentation from Disk

You can install the content from the fulfillment DVD for MSDN Content in either of the following ways.

For individuals installing to their local computers:

  1. Insert the DVD in the DVD drive for your computer.
  2. On the menu bar for Visual Studio 2012, choose Add and Remove Help Content. Help Viewer 2.0 starts, showing the Manage Content tab.
  3. On the Manage Content tab, choose the Disk option button, choose the Browse (…) button, and then navigate to the DVD drive.
  4. Choose the helpcontentsetup.msha file, and then choose the Open button. Under Available Documentation, the list of available content sets from the DVD appears.
  5. Choose the Add link next to the content set that you want to install onto your local computer. The items that you specify appear in the Pending Changes column. You can remove an item from the Pending Changes column by choosing the “X” next to the item.
  6. Choose the Update button.
For enterprise environments:

The fulfillment DVD contains content and an .msha file, which you can copy to a network location that users can access. As an alternative, you can set the client default to point to this location so that each client can manage their own local content. For more information about how to manage Help on users' local computers, see the Help Viewer 2.0 online Administrators Guide.
