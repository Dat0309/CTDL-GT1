MICROSOFT VISUAL STUDIO 2013용추가 구성요소

Microsoft Corporation (또는 거주 지역에 따라 계열사 중 하나)은 귀하에게 본 추가 구성 요소의 사용권을 허여합니다.본 추가 구성 요소가 해당되는 Microsoft Visual Studio 2013("소프트웨어")에 대한 사용권이 있는 경우 본 추가 구성 요소를 사용할 수 있습니다.해당 소프트웨어 사용권이 없는 경우에는 추가 구성 요소도 사용할 수 없습니다.적법하게 사용권이 허여된 소프트웨어의 각 복사본에 본 추가 구성 요소를 사용할 수 있습니다.

다음 사용권 계약에서는 본 추가 구성 요소의 사용 조건에 대해 설명합니다.추가 구성 요소 사용 시 이러한 조건 및 소프트웨어에 대한 사용권 계약이 적용됩니다.두 내용이 상충하는 경우에는 본 추가 구성 요소 사용권 계약이 적용됩니다.

본 추가구성 요소를사용함으로써 귀하는아래의 조항들에 동의하게 됩니다.동의하지 않을경우에는 본추가 구성요소를 사용하지 마십시오.

본사용권 계약을준수하는 경우아래와 같은권리를 행사할수 있습니다.

1.   추가 구성요소에 대한지원 서비스. Microsoft는 www.support.microsoft.com/common/international.aspx 에 기술된 대로 본 소프트웨어에 대한 지원 서비스를 제공합니다.

2.   배포 가능코드. 소프트웨어에는 귀하가 아래 조항을 준수하는 경우 귀하가 개발하는 프로그램에서 배포 권한이 부여되는 코드가 포함되어 있습니다. 본 조항에서 "배포"라는 용어는 제3자가 인터넷을 통해 액세스하기 위한 프로그램 배포를 의미합니다.

a.   JavaScript용 Windows 라이브러리.본 소프트웨어에는JavaScript용 Windows 라이브러리가 포함되어 있습니다. JavaScript용 Windows 라이브러리와 함께 작동하는 귀하의 프로그램에는 본 배포 가능 코드의 다른 조항 외에도 다음 내용이 적용됩니다. JavaScript용 Windows 라이브러리 파일은 귀하의 프로그램에서 Windows 디자인 템플릿과 UI의 외양 및 느낌을 구현하는 데 도움이 될 수 있습니다. 귀하는 내부용으로 개발하거나 귀사가 개발해 제3자에게 배포하는 프로그램에서 수정 사항 없이 이 파일을 복사, 사용 및 배포할 수 있습니다. 귀하는 Windows 스토어 및 Windows Phone 스토어를 통해서만JavaScript용 Windows 라이브러리 파일이 포함되어 있는 프로그램을 제3자에게 배포할 수 있습니다.

b.   소프트웨어에 사용가능한 기능의추가 사용권요구 사항및/또는배포 사용권한.

·       Windows Phone 스토어.귀하의 프로그램을 Windows Phone 스토어에서 제공하려면Windows Phone 스토어 응용 프로그램 공급자 계약의 계약조건을 준수해야 합니다.

·       Bing 지도.소프트웨어에는 맵, 항공 사진 및 하이브리드 이미지 위에 데이터를 표시하는 보고서를 만들기 위해 Bing 지도(또는 후속 모델) 응용 프로그래밍 인터페이스("Bing 지도 API")를 통한 맵, 이미지 및 기타 데이터와 같은 콘텐츠 검색 기능이 포함되어 있을 수 있습니다. 이러한 기능이 포함되어 있는 경우 귀하는 이 기능을 사용하여 동적 문서 또는 정적 문서를 만들고 볼 수 있으며, 이는 소프트웨어에 통합된 액세스 방법 및 수단을 통해서만 가능합니다. Bing 지도 API를 통해 사용할 수 있는 지역 코드, 주소 및 비즈니스 이름 등 업체 정보의 데이터베이스를 복사, 저장, 보관하거나 만들 수 없습니다. Bing 지도 API를 통해 사용 가능한 경우에도 어떤 목적으로든 센서 기반 안내/라우팅을 제공하기 위해 Bing 지도 API를 사용할 수 없으며, 도로 교통 데이터 또는 조감도 이미지(또는 관련 메타데이터)를 사용할 수도 없습니다. Bing 지도 API 및 관련 콘텐츠 사용 시에도 go.microsoft.com/fwlink/?LinkId=21969의 추가 조건이 적용됩니다.

·       추가 매핑 API. 본 소프트웨어에는 맵 및 Bing에서 제공하지 않는 기타 관련 매핑 기능과 서비스("추가 매핑API")가 포함되어 있을 수 있습니다.이 추가 매핑 API에는 추가 조건이 적용되며 해당 추가 매핑 API의 사용 또는 사용량에 따라 Microsoft 및/또는 제3자 공급자에게 요금을 지불해야 할 수도 있습니다.해당 추가 매핑 API를 사용하기 위해 필요한 라이선스 키를 취득할 때 또는 해당 추가 매핑 API 사용에 관련된 설명서를 검토하거나 수령할 때 본 계약조건이 제공됩니다.

·       푸시 알림.Microsoft 푸시 알림 서비스는 중요한 알림을 보내는 데 사용될 수 없으며 그렇지 않으면 의료 장비 또는 상태와 관련된 중요 알림을 포함하여 삶과 죽음의 문제에 영향을 줄 수 있습니다.MICROSOFT는 MICROSOFT 푸시 알림 서비스의 사용 또는MICROSOFT 푸시 알림 서비스의 공급이 중단되거나 오류 없이 사용되거나 실시간 발생 보장에 대한 모든 보증을 명시적으로 부인합니다.

·       Windows.Phone.Speech 네임스페이스 API. 프로그램에서 Windows.Phone.Speech 네임스페이스 API를 통해 음성 인식 기능을 사용하는 데에는 음성 인식 서비스의 지원이 필요합니다.서비스는 인식 시 네트워크 연결이 필요할 수 있습니다(예: 사전 정의된 문법이 필요한 경우). 또한 서비스에서는 서비스를 제공하고 개선하기 위해 음성 관련 데이터를 수집할 수 있습니다.음성 관련 데이터에는 예를 들어 문법 크기 및 문법의 문자열 구와 관련된 정보가 포함될 수 있습니다.

또한 사용자가 휴대폰에서 음성 인식을 사용하려면 우선 특정 사용 조건에 동의해야 합니다. 사용 조건에서는 음성 인식의 사용과 관련된 데이터가 서비스를 제공하고 개선하기 위해 수집되고 사용됨을 사용자에게 통지합니다. 사용자가 사용 조건에 동의하지 않고 응용 프로그램에서 음성 인식을 시도하는 경우 동작하지 않으며 응용 프로그램에 오류를 반환합니다.

·       위치 프레임워크.본 소프트웨어에는 프로그램에서 위치 서비스를 지원할 수 있도록 위치 프레임워크 구성 요소가 포함되어 있을 수 있습니다. 본 계약의 기타 제한뿐만 아니라 귀하는 위치 프레임워크 구성 요소 또는 나머지 소프트웨어 사용 시 모든 관련 지역 법률 및 규정을 준수해야 합니다.

·       장치ID 액세스.본 소프트웨어에는 프로그램이 프로그램에서 실행 중인 장치의 장치 ID에 액세스할 수 있도록 하는 구성 요소가 포함되어 있을 수 있습니다.본 계약의 기타 제한뿐만 아니라 귀사는 장치 ID 액세스 구성 요소 또는 나머지 소프트웨어 사용 시 모든 관련 지역 법률 및 규정을 준수해야 합니다.

·       PlayReady 지원.본 소프트웨어에는Microsoft PlayReady 콘텐츠 액세스 기술이 포함된 Windows Phone Emulator가 포함되어 있습니다.콘텐츠 소유자는Microsoft PlayReady 콘텐츠 액세스 기술을 사용하여 저작권의 보호를 받는 콘텐츠를 포함한 지적 재산권을 보호합니다. 본 소프트웨어는 PlayReady 기술을 사용하여 PlayReady로 보호되는 콘텐츠 및/또는Microsoft DRM(디지털 권한 관리)으로 보호되는 콘텐츠에 액세스합니다.Microsoft는 (i) PlayReady 기술의위반 또는 잠재적 위반이 발생하는 경우, (ii) 사전 예방적 견고성 강화 및 (iii) 소프트웨어에서콘텐츠 사용에 대한 제한을 제대로 이행하지 못해 콘텐츠 소유자가 해지를 요구하는 경우 등(이에 국한되지 않음)의 이유로PlayReady로 보호되는 콘텐츠를 이용하는 소프트웨어의 기능을 해지하도록 결정할 수 있습니다. 해지되는 경우 보안되지 않은 콘텐츠 또는 다른 콘텐츠 액세스 기술로 보호되는 콘텐츠에 영향을 미쳐서는 안 됩니다.콘텐츠 소유자는 귀사가 콘텐츠에 액세스할 때 PlayReady를 업그레이드하도록 요구할 수 있습니다.업그레이드하지 않는 경우 업그레이드가 필요한 콘텐츠에 액세스할 수 없고 다른 운영 체제 업데이트 또는 업그레이드를 설치할 수 없습니다.

·       Microsoft Advertising SDK. 정확한 위치 데이터 또는 그 밖의 사용자 관련 데이터(예: 사용자 식별자, 프로파일 데이터, 행동으로 추적한 사용자 데이터 등)를 Windows Phone용 Microsoft Advertising SDK 또는 Microsoft Advertising SDK에 전달할 경우 귀사의 프로그램은 (a) 사용자 관련 정보를 수집 및 사용하고 Microsoft가 광고에 사용할 수 있도록 이 정보를Microsoft에 제공할 것임을 최종 사용자에게 알리고, (b) 이를 위해 소프트웨어 및/또는 응용 프로그램 다운로드 시 사용자의 명시적인 동의를 받아야 합니다(예: 사용자가 "동의" 또는"설치" 단추를 클릭해야 함). 또한 귀사는 (a) Windows Phone 및 Windows 스토어용 인증 및 기타 요구 사항을 준수하고, (b) 사용자 데이터 수집 및 사용 시 Microsoft 개인정보취급방침과 기타 정책을 준수하고, (c) Microsoft가 생성하거나 귀사에게 제공하는 사용자 식별자를 귀사의 서비스 사용의 Microsoft 광고 서비스에 전달하기 위한 것이 아닌 다른 목적으로 그러한 식별자를 수집하거나 사용하지 않고, (d) 추가 구성 요소에 대한 지원 서비스에서 사용자가 Microsoft의 광고를 선택 해제할 수 있는 링크를 귀사의 개인정보취급방침 및/또는 사용권 계약에 제공할 것에 동의합니다.Microsoft는 www.support.microsoft.com/common/international.aspx 에 기술된 대로 본 소프트웨어에 대한 지원 서비스를 제공합니다.

3.   기타 WINDOWS 구성 요소.본 소프트웨어에는Microsoft .NET Framework 소프트웨어, Microsoft Build 기술 및 패키징 기술과 관련된 특정 .dll 파일, Windows 컨트롤, 리소스 해석과 렌더링 기술, Windows Identity Foundation, Windows 2008용WWS(Windows Web Services) 및JavaScript용 Windows 라이브러리 구성 요소가 포함되어 있습니다. 본 소프트웨어 구성 요소는 Windows의 일부이며 별도의 Windows 사용권 계약에 따라 또는 Windows 사용권의 추가 구성 요소로 귀하에게 사용권이 허여됩니다. 본 Windows 구성 요소 사용 시에는 해당 Windows 제품의 제품 수명 주기 지원 및 사용권 계약이 적용됩니다.
